# Roadmap

## v0.1 MVP

- PRD and task folder with rubrics for agent execution
- npm/npx CLI
- deterministic collectors
- evidence adapter contract
- evidence map as source of truth
- redaction scanner
- claim scoring
- compact briefs
- agent skill
- Markdown/HTML one-project showcase export
- outputs: `CASE_STUDY.md`, `SHOWCASE_BRIEF.md`, `PAGE_CONTENT.md`, `CV_BULLETS.md`, `INTERVIEW_NOTES.md`, `SHOWCASE_PAGE.html`
- dogfood/release checklist to demonstrate data engineering, AI engineering, and digital project management
- redacted private-project example notes for safe manual context
- `npm pack --dry-run` package validation before publishing

## v0.2

- MCP server
- `pow_get_evidence_for_claim`
- `pow_export_showcase`
- better adapter imports
- optional local LLM/Ollama workflow
- better public-safety review

## v0.3

- GitHub PR adapter
- Linear/Jira adapter
- demo transcript parser
- screenshot captions
- richer static showcase template
- optional diagram generation from architecture evidence

## Later
